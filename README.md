Opis sposobu przetestowania funkcjonalności:
Uruchamiamy kod, np. w Visual Studio 2022
Wyświetla nam się strona, wybieramy jedną z zakładek, np. Gry.
Wyświetla nam się lista dodanych już gier (lub po prostu pusta lista jeżeli żadne gry nie były dodane).
Możemy dodać nową grę, podajemy wtedy jej nazwę i opis, id nadaje się samo, a gra staje się widoczna w tabeli.
Utworzoną wcześniej grę możemy edytować, sprawdzać jej szczegóły lub ją usunąć.
Możemy zmieniać to jaką tabelę wyświetlamy, używając nawigacji w górnej części strony.
